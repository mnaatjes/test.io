# Reproduction Feature Notes

## Finding Reproductions
There are four ways to locate available reproductions:
1. **Dashboard:** Displays requests for bugs already approved by the Team Leader (TL).
2. **Inside Bug Reports:** Look for the "Start Reproduction" button within other testers' reports.
3. **Similar Bug Reports List:** While filing a report, if a similar bug appears on the right, click the reproduction button there.
4. **Bugs List:** Activate the "Reproductions available" toggle switch in the Test Cycle bug list.

## Submitting a Reproduction
After opening the form:
1. State if you can reproduce the bug (Positive or Negative).
2. Select the used environment.
3. Upload the required screencast.
4. Click "Create reproduction".

## Removing a Reproduction
1. Go to the **My Reproductions** tab in the Test Cycle header.
2. Open the specific reproduction's detail page.
3. Click **Delete** in the action bar at the bottom.
