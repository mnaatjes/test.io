# Screenshots Reference Notes

**Last Updated:** 2026-03-29

## Core rules
### Usage
Use for static bugs (Visual/Content). Functional bugs need video.

### Visibility
URL field must be visible on website screenshots.

### Edit rule
Highlight the bug on the screenshot.


## Tools by os
### Windows
* Snipping Tool
* Bandicam (Image Capture with drawing mode)

### Macos
* Full Screen: cmd + Shift + 3
* Section: cmd + Shift + 4

### Android
* Home + Power (Older)
* Power + Volume Down (Newer)
* App: ScreenMaster

### Ios ipad
* Home + Power OR Side + Volume Up
* Use Markup tools to edit/highlight immediately.
